PC에서 업로드 → 서버 저장 · 폰에서 같은 반 불러와 연락처 추가

📋 엑셀 만들기 안내 (항상 동일)

1

인트라넷 로그인 → 종합정보 → 교육과정 → 해당 반 선택

2

아래 그림처럼 번호 1부터 학생 정보가 있는 끝까지 드래그 후 복사 (Ctrl+C)

🔍 클릭하면 설명·사진 크게 보기
3

아래 그림처럼 엑셀 A열 1행에 붙여넣기 (Ctrl+V) 후 엑셀 저장

🔍 클릭하면 설명·사진 크게 보기
4

저장한 엑셀을 위 「연락처 엑셀 업로드」로 올린 뒤, 폰으로 접속해 위 「연락처 그룹」 버튼을 눌러 번호순으로 연락처를 추가합니다.
