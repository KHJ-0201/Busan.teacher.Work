엔진오일

교체 주기 · 색상 확인

엔진오일 — 쉽게 이해하기

엔진오일은 엔진 안에서 부품을 부드럽게 돌려 주는 기름입니다.
사람 몸에 기름이 필요하듯, 차에도 꼭 필요합니다.

  • 엔진이 돌아갈 때 마찰과 열을 줄여 줍니다
  • 오일이 부족하거나 오래되면 → 앞에서 본 오일 경고등이 켜질 수 있습니다
  • 양과 색만 가끔 확인해도 큰 고장을 예방할 수 있습니다
다음부터는 보닛을 열고 직접 확인하는 방법을 배워 봅니다.

엔진오일-보닛(엔진룸) 열기

1단계

운전석 왼쪽 아래 레버를 당깁니다.
'쿵' 소리가 나면 1차 개방!

2단계

차 앞으로 나가 보닛 틈새를 확인합니다.
손을 넣을 만큼 열려 있어야 합니다.

3단계

틈새 안쪽 안전 걸쇠를
손가락으로 옆으로 밉니다.

4단계

보닛을 들어 올린 뒤 지지대(꼬챙이)를
걸쳐 고정합니다.

닫을 때: 지지대를 먼저 빼고, 보닛을 30cm 정도 높이에서 놓아 '툭' 소리가 나게 닫으세요.

엔진오일 — 확인 방법

  • 확인 방법: 보닛을 연 뒤 오일 게이지(막대)를 꺼내 닦고, 다시 넣었다 빼서 확인

엔진오일 — 색상 & 교체 주기

  • 색상 기준:
    • 맑은 호박색 → 양호
    • 검고 탁함 → 교체 시기
  • 교체 주기: 약 5,000~10,000km 또는 6~12개월마다
    (차량·운행 환경에 따라 다름)
오일은 직접 교환보다 정비소에서 받으시는 것을 권장합니다!

냉각수

양·누수 점검 · 부동액 혼합

냉각수 — 양·누수 점검

엔진을 적정 온도로 유지하는 냉각수입니다.
부족하면 엔진 과열로 큰 고장이 날 수 있습니다.

  • 확인 시기: 6개월~1년마다, 또는 냉각수 경고등 점등 시
  • 확인 방법: 엔진이 차갑고 보닛을 연 뒤, 부동액(냉각수) 탱크의 MIN ~ MAX 눈금 확인
  • 뚜껑 색: 보통 분홍·빨간색 뚜껑 (차종마다 다를 수 있음)
  • 누수 확인: 주차 자리에 색깔 있는 액체가 고이면 점검 필요
냉각수는 엔진이 뜨거울 때 뚜껑을 열면 위험합니다!
반드시 엔진이 식은 후 확인하세요.

냉각수 — 부동액 혼합

부동액은 보통 물과 섞어 냉각수로 사용합니다.

  • 혼합 비율: 제품·차종마다 다름 — 라벨의 권장 비율 확인 (예: 물 50% + 부동액 50%)
  • 주의: 서로 다른 종류의 부동액을 섞지 마세요
  • 보충: 부족할 때는 같은 제품으로 천천히 보충
냉각수·부동액은 엔진이 식은 후 보충하세요!

배터리

방전 예방 · 겨울철 관리

배터리

배터리 역할

  • 시동: 시동 모터·점화 장치에 전력 공급
  • 전기 장치: 전조등·계기판·오디오 등 전기를 사용할 때 보조
  • 충전: 운행 중 발전기가 배터리를 다시 충전합니다

배터리 옆면·상단 라벨에서 아래 정보를 확인할 수 있습니다.

  • 전압(V): 12V 배터리 전압
  • 용량(Ah): 90AH 배터리 크기·성능 참고
  • 160MIN: 예비 용량(분) — 방전 시 버틸 수 있는 시간 참고
  • 극성(+/−): R 연결 방향 확인 (교체 시 중요)
라벨 정보가 흐리거나 지워졌으면 정비소에서 확인하세요!

배터리 — 방전 예방 & 겨울철 팁

  • 보통 2~3년 사용 후 교체 권장
  • 시동이 느리거나 전조등이 어두워지면 → 점검!
  • 계기판 배터리 경고등 → 충전·교체 확인
  • 장기간(2주+) 미운행 시 → 주 1회 20분 이상 시동

❄️ 겨울철 특별 관리

  • 추위에 배터리 성능 30%↓ — 방전 주의
  • 출발 전 30초 예열 후 출발 (긴 예열 불필요)

타이어

공기압 · 100원 동전 마모 확인

타이어

타이어 옆면에는 차량에 맞는 사이즈와 제조 정보가 적혀 있습니다.

  • 사이즈: 예) 205/55R16 — 폭 · 편평비 · 휠(림) 크기
  • 하중·속도: 숫자+알파벳 — 허용 하중 · 최고 속도 등급
  • 제조 주차: DOT 뒤 4자리 — 앞 2자리=주, 뒤 2자리=년
  • 교체 참고: 마모가 없어도 보통 5~6년이면 교체 검토
타이어 사이즈는 차량 설명서 또는 운전석 문 스티커와 같은지 확인하세요!

타이어 점검 — 100원 동전 활용법

  • 공기압: 주유소 에어 게이지로 월 1회 확인
    타이어 옆면에 적힌 숫자(kPa) 참고
  • 마모 확인: 100원 동전을 홈에 끼워 넣어 보세요

브레이크

소리 · 페달 감각 · 브레이크 오일

브레이크

브레이크는 운전 중 속도를 줄이고 차를 멈추게 하는 장치입니다.

  • 페달을 밟으면 브레이크 패드가 디스크를 눌러 마찰로 제동합니다
  • 앞·뒤 바퀴에 연결되어 균형 있게 차량을 멈춥니다
  • 배력장치가 페달 힘을 도와, 가볍게 밟아도 충분히 제동됩니다
브레이크는 안전과 직결되는 가장 중요한 장치입니다!

브레이크 점검 — 소리와 발의 감각

브레이크는 눈으로 보기 어렵습니다.
소리와 페달 감각으로 확인하세요!

  • 소리: '끼익—' 금속 마찰음 → 패드(마찰재) 교체 시기
  • 페달: 밟을 때 깊게 들어가거나 떨림 → 점검 필요
  • 브레이크등: 계기판 경고등 점등 → 즉시 정비소
브레이크는 안전과 직결됩니다. 이상하면 바로 정비소에 가세요!

브레이크 오일

브레이크 오일(유압)은 페달 힘을 각 바퀴까지 전달하는 액체입니다.

  • 역할: 오일이 압력을 전달해 패드가 디스크를 누릅니다
  • 상태 확인: 맑으면 양호, 어둡거나 탁하면 교체 시기
  • 수분 점검: 오일에 수분이 섞이면 제동력이 떨어집니다 — 수분 테스터기로 확인
  • 교체 주기: 보통 2~4년마다 (차종·사용 환경에 따라 다름)
브레이크 오일 교환은 정비소에서 받으시는 것을 권장합니다!

소모품

와이퍼+워셔액 · 에어컨+필터 · 각종 전구

소모품 — 와이퍼

비·눈 오는 날 앞유리를 닦아 시야를 확보하는 소모품입니다.

  • 교체 주기: 6개월~1년
  • 고무가 굳거나 닳으면 잘 닦이지 않습니다

교체 증상 — 자국 · 안 닦임 · 소리·떨림 → 바로 교체!

소모품 — 와이퍼 교체 방법

소모품 — 워셔액

워셔액은 앞유리를 닦는 세정액입니다.
파란 뚜껑 주입구에서 직접 보충할 수 있습니다.

  • 파란 뚜껑 — 워셔액 주입구 (엔진룸 앞쪽)
  • MAX 선까지 천천히 보충
  • 겨울엔 동결 방지 워셔액 사용
뚜껑을 열 때 주변에 먼지가 들어가지 않게 주의하세요!

소모품 — 에어컨

에어컨은 여름 냉방·겨울 난방에 쓰는 장치입니다.

  • 시즌 전 냉·난방 작동을 미리 확인하세요
  • A/C 버튼으로 에어컨을 켜고 끕니다
  • 바람이 약하거나 냄새가 나면 → 필터 점검

소모품 — 에어컨 필터

에어컨 필터는 들어오는 공기의 먼지·꽃가루를 걸러줍니다.

  • 흐름: 실외·실내 공기 → 필터 → 블로어모터 → 실내
  • 교체: 10,000~20,000km (보통 1년마다)
  • 바람 약함·냄새·먼지 → 교체 시기
  • 깨끗한 필터와 비교해 누렇게 변하면 교체!

소모품 — 등화장치 · 전구

차에는 앞·뒤·좌우마다 역할이 다른 등이 있습니다.

  • 위치: 전조등·방향지시등(앞) · 미등·제동등·후진등·번호등(뒤) · 실내등
  • 종류: 전조등 · 안개등 · 방향지시등 · 제동등 · 후진등 · 번호등 등
  • 전구: 등마다 규격이 다름 (H4, H7, W5W 등) — 교체 시 같은 번호 확인
  • 벽에 빛 비춰 양쪽 밝기 비교 · 한쪽만 어두우면 교체
